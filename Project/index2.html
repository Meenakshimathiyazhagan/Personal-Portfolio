INSIGHT STREAM – NAVIGATE THE NEWS IS A FULL-STACK WEB APPLICATION

Insight Stream Navigate the News is a full-stack web application developed as my individual placement project for the Edunet Foundation. I collaborated with four teammates to build a platform that helps editors, analysts, and readers filter the overwhelming volume of daily headlines into a clear, actionable stream of insights. Inspired by research from the Media Insight Project — a collaboration between the American Press Institute and The AP NORC Center — this platform enables organizations to analyze where audiences engage, where they lose interest, and how they can optimize strategies for better reader retention and sustainable revenue. At its core is an Automated Insights Feed powered by live APIs that pull real-time news, traffic spikes, and keyword trends, displaying each item with a quick recommendation for immediate action. Users can personalize their experience by pinning topics, rearranging widgets, switching themes, and collapsing panels for better focus. Built-in tools like commenting, sharing via email, and workspace exports support team collaboration. From a technical standpoint, the application is built using the MERN stack. React, React Router, and Tailwind CSS create a fully responsive and dynamic front-end using Grid and Flexbox layouts. React Context API stores persistent user preferences and bookmarks, while useState manages fast, responsive interactions. The backend is developed using Node.js and Express.js, which aggregate data from multiple sources, apply filters, and ensure secure API handling through CORS.

MongoDB Atlas is used for storing user data, saved content, and analytical logs. Continuous integration via GitHub Actions runs tests, linters, and performance audits, while Docker deployments to Render ensure stable and rollback-capable hosting. Performance metrics are tracked using Web Vitals, and error monitoring is managed with Sentry. Planned future enhancements include integrating OpenAI models to summarize articles, apply sentiment analysis, and personalize content through a recommendation engine. UX updates will include voice search, offline support, multilingual translation, live polls, and real-time news tickers via WebSockets. Monetization features such as premium tiers, affiliate content, and respectful native ads are also in consideration. Known technical issues such as rate limits, infinite scroll bugs on legacy devices, large image loads, and minor SEO challenges are documented and resolved through agile sprints and regression testing. Insight Stream demonstrates how strong engineering and thoughtful design can transform chaotic information into clarity. By combining AI, full-stack development, and user-focused features, this application proves that meaningful, personalized news delivery can benefit both organizations and the public.
